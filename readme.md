# Репозиторій Кубічного Вікі

## Проєкт: CubedVij Wiki (VitePress)

Цей репозиторій містить документацію, зібрану та опубліковану за допомогою **VitePress**.

### Вимоги

- **Node.js** (рекомендовано LTS)
- **npm** або **pnpm** або **yarn**

### Встановлення

```bash
npm install
```

### Запуск локального сервера

```bash
npm run docs:dev
```

Після запуску сайт буде доступний за адресою, яку виведе термінал (зазвичай `http://localhost:5173`).

### Збірка (build)

```bash
npm run docs:build
```

### Попередній перегляд зібраної версії

```bash
npm run docs:preview
```

### Структура документації

Типова структура (може відрізнятися у вашому проєкті):

```text
.
├─ docs/
│  ├─ .vitepress/
│  │  └─ config.mts (або config.ts)
│  ├─ index.md
│  └─ ...
└─ package.json
```

- Усі сторінки документації — це файли `*.md` у папці `docs/`.
- Налаштування сайту — в `docs/.vitepress/config.*`.

### Додавання сторінок

1. Створіть новий файл, наприклад: `docs/guide.md`
2. Додайте посилання на нього в навігації/сайдбарі у `docs/.vitepress/config.*`

### Корисні посилання

- Документація VitePress: https://vitepress.dev/
- Markdown довідка: https://www.markdownguide.org/
