# ⚙️ Список корисних команд

Нижче наведено список корисних команд, які допоможуть вам отримати повний досвід гри на нашому сервері.

### **Реєстрація та Авторизація**
* `/register <пароль> <пароль>` — Реєстрація при першому вході.  
* `/login <пароль>` — Вхід у гру.  
* `/auth update <пароль>` — Зміна пароля.

### **Скіни**
* `/skin <нік>` — Встановити скін ліцензійного гравця (наприклад: /skin Notch).  
* `/skin url <посилання.png>` — Встановити скін за прямим посиланням.

### **Голосовий Чат**
* **Як користуватись:** Натисніть **V**, щоб відкрити налаштування (якщо мод встановлено у вас на клієнті).  
* Якщо мода немає — ви просто не будете чути голосовий чат, але зможете грати.

### **Кастомні Крафтии**
* `/polydex` — щоб переглянути рецепти крафтів прямо на сервері (аналог JEI).  
* `/calc <формула>` — калькулятор для швидких обчислень (наприклад: /calc 64*5).

### **Могили**
* `/graves` — Показати координати ваших могил.
