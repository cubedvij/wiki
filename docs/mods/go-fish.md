# Риболовля

::: tip 
**Go Fish** — це мод для Fabric, покликаний покращити риболовлю в Minecraft. Надихаючись *Stardew Valley* та *Terraria*, він пропонує більш захопливий досвід, який винагороджує вас за спокійний відпочинок на пляжі з вудкою в руках.
:::

## Основний контент

Go Fish додає звичайну рибу, яку можна зловити будь-де у кожному вимірі, а також специфічну для біомів рибу, яка водиться лише в певних місцях.

![Go Fish](/images/mods/goon-fishing.png)

### Верхній світ (Overworld)

## **Глобальні ящики (Global Crates):**

* Дерев'яний ящик (Wooden Crate)
* Залізний ящик (Iron Crate)
* Золотий ящик (Golden Crate)
* Діамантовий ящик (Diamond Crate)

**Крижані біоми (Icy):**

* Риба-бурулька (Icicle Fish)
* Риба-сніжка (Snowball Fish)
* Заморожений ящик (Frosted Crate)

**Болото (Swamp):**

* Слизориба (Slimefish)
* Лататтєва риба (Lilyfish)
* Слизовий ящик (Slimey Crate)

**Меса/Пустка (Mesa):**

* Терра-риба (Terrafish)

**Рівнини та Ліси (Plains & Forests):**

* Морквяний короп (Carrot Carp)
* Дубова риба (Oakfish)
* Ящик з припасами (Supply Crate)

**Різне (Misc):**

* *Під час повного місяця:* Місячна риба (Lunarfish), Галактична морська зірка (Galaxy Starfish), Зоряний лосось (Starry Salmon), Туманна риба-меч (Nebula Swordfish), Астральний ящик (Astral Crate).
* *Залежно від погоди:* Дощовий окунь (Rainy Bass), Грозовий окунь (Thundering Bass), Хуртовинний окунь (Blizzard Bass).
* *Специфічне для локації:* Хмарний краб (Cloudy Crab).

### Незер (The Nether)

**Глобально:**

* Димчастий лосось (Smokey Salmon)
* Магмова тріска (Magma Cod)
* Риба-кістка (Bonefish)
* Обсидіановий палтус (Obsidian Halibut)
* Базальтовий окунь (Basalt Bass)
* Шипориба (Spikerfish)
* Вогняний ящик (Fiery Crate)

**Базальтові дельти (Basalt Deltas):**

* Позолочений чорнокам'яний короп (Gilded Blackstone Carp)
* Позолочений чорнокам'яний ящик (Gilded Blackstone Crate)
* Чорнокам'яна форель (Blackstone Trout)

**Долина душ (Soul Sand Valley):**

* Лосось душ (Soul Salmon)
* Ящик душ (Soul Crate)

### Енд (The End)

* Запечена енд-риба (Baked Endfish)
