## Правила спільноти
Щоб забезпечити дружню та позитивну атмосферу для всіх учасників, будь ласка, дотримуйтесь наступних правил:
1. Поважайте інших учасників спільноти.
2. Уникайте образливих висловлювань та поведінки.
3. Не поширюйте спам або рекламу без дозволу адміністрації.
4. Дотримуйтесь правил сервера Кубічний Двіж.
